TM2-xaseco2-plugin for showing an event-banner + coundown

INSTALLATION:
1) put the event.php-file in the xaseco2/plugins-folder
2) in the xaseco2/plugins.xml add the line:
   <plugin>plugin.event.php</plugin>
3) put the event.xml-file in the xaseco2-folder (you can adapt some style-things in this file easily and you can indicate shown picture, pointed-to url and countdown-end-date)
4) restart xaseco2
